Studio Library Admin

Manage Creator Studio backgrounds, graphics & product templates. Changes go live ~2 minutes after saving.

Sign in

Create a fine-grained token at github.com/settings/personal-access-tokens/new: choose Only select repositories → printncraftco.com, and under Repository permissions set Contents → Read and write. Nothing on this page works without a valid token.

Collections

Upload images

Blueprint = the guide image with the red print zone. Shading = the realistic overlay. Both should match the product's wrap dimensions (same aspect ratio).

Files

Hidden items stay in storage but disappear from the studio. Deletes are permanent (though recoverable from git history). After any change, GitHub rebuilds the library index and republishes the site — allow a couple of minutes, then refresh the studio.

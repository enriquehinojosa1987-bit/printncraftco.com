Studio Library Admin

Manage Creator Studio backgrounds, graphics & product templates. Changes are staged locally, then published in one shot.

Sign in

Create a fine-grained token at github.com/settings/personal-access-tokens/new: choose Only select repositories → printncraftco.com, and under Repository permissions set Contents → Read and write. Nothing on this page works without a valid token.

Collections

Upload images

Blueprint = the guide image with the red print zone. Shading = the realistic overlay. Both should match the product's wrap dimensions (same aspect ratio).

Files

Nothing touches the live site until you hit Publish — stage as many uploads, deletes, and hide/shows as you want, then push them as a single update (one build, ~2 min). Hidden items stay in storage but disappear from the studio. Deletes are permanent after publishing (though recoverable from git history).
